About LyricView Pro V2

Built to help churches present worship content more smoothly and professionally.

LyricView Pro V2 is a desktop church presentation software created for worship leaders, media teams, and churches that need a simple but powerful tool for displaying lyrics, Bible verses, and media during worship services.


It focuses on speed, clean presentation, modern design, and practical features that work well in real ministry settings.
